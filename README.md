# brady0127.github.io

My personal academic homepage — built with pure HTML, CSS, and vanilla JS.

## Structure

```
├── index.html          # Homepage
├── research.html       # Publications & research
├── projects.html       # Projects & open-source work
├── blog.html           # Blog posts
├── others.html         # Talks, teaching, reading, misc
├── assets/
│   ├── css/
│   │   └── style.css   # All styles
│   ├── js/
│   │   └── main.js     # Mobile nav + scroll animations
│   └── img/
│       └── (your portrait photo here)
└── README.md
```

## Setup

1. Clone this repo (or fork it)
2. Replace placeholder content with your own
3. Add your portrait photo to `assets/img/` and update the `<img>` tag in `index.html`
4. Update social links in the hero section
5. Push to GitHub — it will auto-deploy at `https://brady0127.github.io`

## Customization

- **Colors**: Edit CSS variables in `:root` at the top of `style.css`
- **Fonts**: Change the Google Fonts import in each HTML `<head>`
- **Content**: All content is in the HTML files — no build step required
